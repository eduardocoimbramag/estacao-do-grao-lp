# Documentação: Ajustes Adicionais da Seção "Nossos Serviços" (LAYDESK2) - Segunda Rodada

## Objetivo
Aplicar ajustes adicionais na seção "Nossos Serviços" (Section 3) no layout `laydesk2` para:
1. Eliminar o overflow restante no carrossel
2. Reduzir a largura dos cards do carrossel
3. Reduzir ainda mais os espaçamentos internos dos cards
4. Colar o subtítulo ao título (espaçamento mínimo)

---

## Problemas Identificados (Segunda Rodada)

1. **Overflow ainda presente** - O conteúdo do carrossel ainda ultrapassa os limites da seção
2. **Largura dos cards excessiva** - Os cards estão muito largos, ocupando muito espaço horizontal
3. **Espaçamentos internos ainda grandes** - Os espaçamentos dentro dos cards precisam ser reduzidos ainda mais
4. **Subtítulo muito distante do título** - O espaçamento entre "Nossos Serviços" e o subtítulo precisa ser mínimo (colado)

---

## Estado Atual do CSS

### Valores Atuais (após primeira rodada de ajustes):

- Header: `margin-bottom: 1rem`, `margin-top` (título→subtítulo): `0.25rem`
- Cards: `lg:min-w-[33.333%]` (Tailwind - 33.33% da largura)
- Card content: `padding-left/right: 0.75rem`, `padding-bottom: 1.25rem`
- Card content interno: `margin-top: 0.375rem`
- Imagem container: `padding: 0.75rem`
- Imagem: `height: 7.5rem`
- CTA: `padding-top: 0.5rem`

---

## Plano de Ajustes Adicionais

### 1. **Eliminar Overflow Restante no Carrossel**

**Problema:** Ainda há overflow vertical no carrossel.

**Análise:** O overflow pode estar sendo causado por:
- Padding-bottom do container ainda muito grande
- Altura dos cards (imagem + conteúdo) ainda muito grande
- Espaçamentos internos acumulados

**Solução:**
- Reduzir ainda mais o `padding-bottom` do container (de `1.5rem` para `0.75rem`)
- Reduzir o `padding-bottom` do container do carrossel (de `0.75rem` para `0.5rem`)
- Garantir que a altura total dos cards não ultrapasse o espaço disponível

**CSS:**
```css
/* [SEC3] Container - Padding bottom reduzido (LAYDESK2)
   Impacto: reduz espaço inferior, liberando mais espaço para o carrossel */
.laydesk2-servicos-container {
  padding-bottom: 0.75rem !important; /* REDUZIDO de 1.5rem */
}

/* [SEC3] Container do carrossel - Padding bottom reduzido (LAYDESK2)
   Impacto: reduz espaço inferior do carrossel */
.laydesk2-servicos-carousel-container {
  padding-bottom: 0.5rem !important; /* REDUZIDO de 0.75rem */
}
```

---

### 2. **Reduzir Largura dos Cards**

**Problema:** Os cards estão muito largos (33.33% da largura).

**Análise:** No Tailwind, os cards usam `lg:min-w-[33.333%]`, que em uma tela de 1024px-1279px pode resultar em cards muito largos. Precisamos forçar uma largura menor via CSS.

**Solução:**
- Sobrescrever a largura mínima dos cards via CSS
- Usar `max-width` para limitar a largura máxima de cada card
- Reduzir de ~33.33% para ~28-30% da largura disponível

**CSS:**
```css
/* [SEC3] Cards - Largura reduzida (LAYDESK2)
   Impacto: reduz a largura dos cards, permitindo melhor distribuição e menor ocupação horizontal */
.laydesk2-servicos-card {
  min-width: 28% !important; /* REDUZIDO de 33.333% (Tailwind) */
  max-width: 28% !important; /* limita a largura máxima */
}
```

---

### 3. **Reduzir Espaçamentos Internos dos Cards (Mais Agressivo)**

**Problema:** Os espaçamentos internos dos cards ainda estão grandes.

**Análise:** Precisamos reduzir ainda mais:
- Padding do container de imagem
- Padding do conteúdo do card (horizontal e vertical)
- Espaçamento interno entre elementos (`margin-top`)
- Padding do CTA

**Solução:**
- Reduzir padding do container de imagem (de `0.75rem` para `0.5rem`)
- Reduzir padding horizontal do conteúdo (de `0.75rem` para `0.5rem`)
- Reduzir padding-bottom do conteúdo (de `1.25rem` para `0.875rem`)
- Reduzir espaçamento interno (de `0.375rem` para `0.25rem`)
- Reduzir padding-top do CTA (de `0.5rem` para `0.25rem`)

**CSS:**
```css
/* [SEC3] Container da imagem - Padding ainda mais reduzido (LAYDESK2)
   Impacto: reduz o espaço ao redor da imagem */
.laydesk2-servicos-image-container {
  padding: 0.5rem !important; /* REDUZIDO de 0.75rem */
}

/* [SEC3] Conteúdo do card - Padding ainda mais reduzido (LAYDESK2)
   Impacto: reduz o espaço interno do card */
.laydesk2-servicos-card-content {
  padding-left: 0.5rem !important; /* REDUZIDO de 0.75rem */
  padding-right: 0.5rem !important; /* REDUZIDO de 0.75rem */
  padding-bottom: 0.875rem !important; /* REDUZIDO de 1.25rem */
}

/* [SEC3] Conteúdo do card - Espaçamento interno ainda mais reduzido (LAYDESK2)
   Impacto: aproxima ainda mais os elementos de texto dentro do card */
.laydesk2-servicos-card-content > * + * {
  margin-top: 0.25rem !important; /* REDUZIDO de 0.375rem */
}

/* [SEC3] CTA do card - Padding ainda mais reduzido (LAYDESK2)
   Impacto: reduz o espaço acima do botão CTA */
.laydesk2-servicos-card-cta {
  padding-top: 0.25rem !important; /* REDUZIDO de 0.5rem */
}
```

---

### 4. **Colar Subtítulo ao Título**

**Problema:** O subtítulo ainda está distante do título "Nossos Serviços".

**Análise:** Atualmente o `margin-top` entre título e subtítulo está em `0.25rem`. Para "colar", precisamos reduzir para `0` ou `0.125rem` (mínimo).

**Solução:**
- Reduzir `margin-top` entre título e subtítulo para `0` (ou `0.125rem` para respiro mínimo)

**CSS:**
```css
/* [SEC3] Header - Espaçamento título→subtítulo mínimo (LAYDESK2)
   Impacto: cola o subtítulo ao título */
.laydesk2-servicos-header > * + * {
  margin-top: 0 !important; /* REDUZIDO de 0.25rem (COLA) */
}
```

## Notas Importantes

1. **Apenas LAYDESK2**: Todas as mudanças são exclusivas para o layout `laydesk2` (media query `1024px - 899px`). Não alterar outros layouts.

2. **Ajustes Progressivos**: Estas são mudanças adicionais à primeira rodada. Os valores são reduzidos em relação aos valores já ajustados anteriormente.

3. **Anti-Overflow Prioridade**: A eliminação do overflow é a prioridade máxima. Se necessário, reduzir ainda mais alguns valores.

4. **Largura dos Cards**: A redução de `33.333%` para `28%` pode parecer pequena, mas em uma tela de ~1200px, isso representa ~64px de redução por card, o que pode ajudar significativamente.

5. **Subtítulo Colado**: O `margin-top: 0` pode fazer o subtítulo ficar completamente colado. Se o visual ficar muito apertado, considerar `0.125rem` como alternativa mínima.

6. **Teste Visual**: Após aplicar as mudanças, testar em diferentes resoluções dentro do range `laydesk2` para garantir que tudo está enquadrado corretamente e sem overflow.
