# Documentação: Ajustes da Seção "Nossos Serviços" (LAYDESK2)

## Objetivo
Ajustar a seção "Nossos Serviços" (Section 3) no layout `laydesk2` para eliminar overflow e enquadrar todo o conteúdo dentro de "1 tela - menu", reduzindo espaçamentos e tamanhos de forma mínima e harmônica.

---

## Problemas Identificados

1. **Overflow no conteúdo do Carrossel** - O carrossel está ultrapassando os limites da seção
2. **Espaçamentos excessivos** - Título muito distante do topo, subtítulo muito distante do título
3. **Distância entre subtítulo e carrossel** - Espaço excessivo entre o header e o carrossel
4. **Tamanho do carrossel** - Cards e imagens muito grandes
5. **Tamanho dos botões de navegação** - Botões e ícones muito grandes

---

## Plano de Ajustes

### 1. **Eliminar Overflow no Carrossel**

**Problema:** O carrossel está ultrapassando os limites da seção.

**Solução:**
- Garantir que `.laydesk2-servicos-section` tenha `overflow: hidden`
- Garantir que `.laydesk2-servicos-container` use `flex: 1` e `min-height: 0` para permitir shrink
- Garantir que `.laydesk2-servicos-carousel-container` tenha `max-height` e `overflow: hidden`

**CSS:**
```css
/* [SEC3] Seção - Anti-overflow (LAYDESK2)
   Impacto: impede que o conteúdo ultrapasse os limites da seção */
.laydesk2-servicos-section {
  overflow: hidden !important; /* FORÇAR - impede overflow horizontal/vertical */
}

/* [SEC3] Container - Flex shrink habilitado (LAYDESK2)
   Impacto: permite que o container encolha se necessário */
.laydesk2-servicos-container {
  min-height: 0 !important; /* permite shrink */
  overflow: hidden !important; /* impede overflow interno */
}

/* [SEC3] Container do carrossel - Limitar altura (LAYDESK2)
   Impacto: garante que o carrossel não ultrapasse a altura disponível */
.laydesk2-servicos-carousel-container {
  max-height: 100% !important;
  overflow: hidden !important;
}
```

---

### 2. **Reduzir Espaçamentos do Header**

**Problema:** Título muito distante do topo, subtítulo muito distante do título.

**Solução:**
- Reduzir `padding-top` do `.laydesk2-servicos-container` (de `1.5rem` para `0.75rem`)
- Reduzir `margin-bottom` do `.laydesk2-servicos-header` (de `1.75rem` para `1rem`)
- Reduzir `margin-top` entre título e subtítulo (de `0.875rem` para `0.25rem`)

**CSS:**
```css
/* [SEC3] Container - Padding top reduzido (LAYDESK2)
   Impacto: aproxima o título do topo da seção */
.laydesk2-servicos-container {
  padding-top: 0.75rem !important; /* REDUZIDO de 1.5rem */
}

/* [SEC3] Header - Margin bottom reduzido (LAYDESK2)
   Impacto: aproxima o header do carrossel */
.laydesk2-servicos-header {
  margin-bottom: 1rem !important; /* REDUZIDO de 1.75rem */
}

/* [SEC3] Header - Espaçamento entre título e subtítulo reduzido (LAYDESK2)
   Impacto: aproxima o subtítulo do título */
.laydesk2-servicos-header > * + * {
  margin-top: 0.25rem !important; /* REDUZIDO de 0.875rem */
}
```

---

### 3. **Reduzir Distância entre Subtítulo e Carrossel**

**Problema:** Espaço excessivo entre o header e o carrossel.

**Solução:**
- Já coberto no item 2 (redução do `margin-bottom` do header)
- Adicionalmente, reduzir `padding-top` do `.laydesk2-servicos-carousel-container` (de `0.75rem` para `0.25rem`)

**CSS:**
```css
/* [SEC3] Container do carrossel - Padding top reduzido (LAYDESK2)
   Impacto: aproxima o carrossel do subtítulo */
.laydesk2-servicos-carousel-container {
  padding-top: 0.25rem !important; /* REDUZIDO de 0.75rem */
}
```

### 4. **Reduzir Tamanho do Carrossel**

**Problema:** Cards e imagens muito grandes.

**Solução:**
- Reduzir altura das imagens (de `10rem` para `7.5rem`)
- Reduzir padding do container de imagem (de `1rem` para `0.75rem`)
- Reduzir padding do conteúdo do card (de `1rem` para `0.75rem` horizontal, de `2rem` para `1.25rem` vertical)
- Reduzir espaçamento interno do conteúdo (de `0.625rem` para `0.375rem`)
- Reduzir tamanhos de fonte (título: de `1.125rem` para `1rem`, subtítulo: de `0.8125rem` para `0.75rem`, descrição: de `0.8125rem` para `0.75rem`)
- Reduzir padding do CTA (de `0.75rem` para `0.5rem`)

**CSS:**
```css
/* [SEC3] Container da imagem - Padding reduzido (LAYDESK2)
   Impacto: reduz o espaço ao redor da imagem */
.laydesk2-servicos-image-container {
  padding: 0.75rem !important; /* REDUZIDO de 1rem */
}

/* [SEC3] Imagem - Altura reduzida (LAYDESK2)
   Impacto: reduz a altura da imagem do card */
.laydesk2-servicos-image {
  height: 7.5rem !important; /* REDUZIDO de 10rem (120px → 90px) */
}

/* [SEC3] Conteúdo do card - Padding reduzido (LAYDESK2)
   Impacto: reduz o espaço interno do card */
.laydesk2-servicos-card-content {
  padding-left: 0.75rem !important; /* REDUZIDO de 1rem */
  padding-right: 0.75rem !important; /* REDUZIDO de 1rem */
  padding-bottom: 1.25rem !important; /* REDUZIDO de 2rem */
}

/* [SEC3] Conteúdo do card - Espaçamento interno reduzido (LAYDESK2)
   Impacto: aproxima os elementos de texto dentro do card */
.laydesk2-servicos-card-content > * + * {
  margin-top: 0.375rem !important; /* REDUZIDO de 0.625rem */
}

/* [SEC3] Título do card - Fonte reduzida (LAYDESK2)
   Impacto: reduz o tamanho do título do card */
.laydesk2-servicos-card-title {
  font-size: 1rem !important; /* REDUZIDO de 1.125rem */
  line-height: 1.5rem !important; /* REDUZIDO de 1.625rem */
}

/* [SEC3] Subtítulo do card - Fonte reduzida (LAYDESK2)
   Impacto: reduz o tamanho do subtítulo do card */
.laydesk2-servicos-card-subtitle {
  font-size: 0.75rem !important; /* REDUZIDO de 0.8125rem */
  line-height: 1rem !important; /* REDUZIDO de 1.125rem */
}

/* [SEC3] Descrição do card - Fonte reduzida (LAYDESK2)
   Impacto: reduz o tamanho da descrição do card */
.laydesk2-servicos-card-description {
  font-size: 0.75rem !important; /* REDUZIDO de 0.8125rem */
  line-height: 1.125rem !important; /* REDUZIDO de 1.25rem */
}

/* [SEC3] CTA do card - Padding reduzido (LAYDESK2)
   Impacto: reduz o espaço acima do botão CTA */
.laydesk2-servicos-card-cta {
  padding-top: 0.5rem !important; /* REDUZIDO de 0.75rem */
}
```

### 5. **Reduzir Tamanho dos Botões de Navegação**

**Problema:** Botões e ícones muito grandes.

**Solução:**
- Reduzir `margin-top` do `.laydesk2-servicos-nav` (de `0.75rem` para `0.5rem`)
- Reduzir `padding` dos botões (de `0.625rem` para `0.5rem`)
- Reduzir tamanho dos ícones SVG (de `18px` para `16px`)

**CSS:**
```css
/* [SEC3] Botões de navegação - Margin top reduzido (LAYDESK2)
   Impacto: aproxima os botões do carrossel */

  margin-top: 0.5rem !important; /* REDUZIDO de 0.75rem */
}

/* [SEC3] Botões de navegação - Padding reduzido (LAYDESK2)
   Impacto: reduz o tamanho dos botões */
.laydesk2-servicos-nav button {
  padding: 0.5rem !important; /* REDUZIDO de 0.625rem */
}

/* [SEC3] Ícones dos botões - Tamanho reduzido (LAYDESK2)
   Impacto: reduz o tamanho dos ícones SVG */
.laydesk2-servicos-nav svg {
  width: 16px !important; /* REDUZIDO de 18px */
  height: 16px !important; /* REDUZIDO de 18px */
}
```

## Localização no CSS

## Notas Importantes

1. **Apenas LAYDESK2**: Todas as mudanças são exclusivas para o layout `laydesk2` (media query `1024px - 1279px`). Não alterar outros layouts.

2. **Mudanças Mínimas**: O objetivo é fazer ajustes sutis para enquadrar o conteúdo sem alterar drasticamente a aparência.

3. **Anti-Overflow**: A prioridade é eliminar qualquer overflow, mesmo que isso signifique reduzir um pouco mais alguns elementos.

4. **Teste Visual**: Após aplicar as mudanças, testar em diferentes resoluções dentro do range `laydesk2` para garantir que tudo está enquadrado corretamente.
